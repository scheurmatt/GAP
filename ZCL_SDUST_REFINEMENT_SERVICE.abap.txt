CLASS zcl_sdust_refinement_service DEFINITION
  PUBLIC FINAL CREATE PUBLIC.

  PUBLIC SECTION.

    TYPES tt_uuid TYPE STANDARD TABLE OF sysuuid_x16 WITH EMPTY KEY.

    CLASS-METHODS refine_mass
      IMPORTING it_uuid TYPE tt_uuid.

ENDCLASS.

CLASS zcl_sdust_refinement_service IMPLEMENTATION.


METHOD refine_mass.

  LOOP AT it_uuid INTO DATA(lv_uuid).

    DATA(lv_task) = |REF_{ sy-tabix }|.

    CALL FUNCTION 'Z_SDUST_REFINE_FM'
      STARTING NEW TASK lv_task
      EXPORTING
        iv_uuid = lv_uuid.

  ENDLOOP.

ENDMETHOD.
  endclass.
